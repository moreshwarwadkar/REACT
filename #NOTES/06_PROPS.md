-----> PROPS :<br/><br/>

- Props are nothing but properties.<br/>
- Props are used to share the data from one component to another component.<br/>
- Props follows unidirectional flow.<br/>
- By default all the props will be store as objects and pass to the child component.<br/>
- Props are immutable, it means once the data is passed from parent component can't be changed in child component.<br/>
- You can send any type of data using props.<br/>
- Child Component can only received the props data but cannot modify/mutate the sender data.<br/><br/>



-----> Difference Between Props and State :<br/>

- Props : It used to pass the data from one component to another component.<br/>
- State : It is the Object and it manage the properties within the component.<br/><br/>

- Props : It is Immutable. <br/>
- State : It is Mutable<br/><br/>

- Props : Props will help you to make the components reusable. <br/>
- State : It is used to handle interactive data that changes over a time.<br/><br/>
