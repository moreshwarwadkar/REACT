---------- JSX ----------</br>

- JSX is Stands for JavaScript XML.</br>
- Which allows JavaScript to Wrap HTML Code.</br>
- It is a Template Language for React JS.</br>
- It looks similar like HTML, but it's not HTML.</br></br>

--> Rules Of JSX :</br>

1) We have to close all the tags including self-closing tags.</br>
2) Use CamelCase for attributes.</br>
3) Instead of "class" we have to use "className".</br>
   Instead of "for" we have to use "htmlFor".</br>
4) We need to wrap Sibling Elements in a Parent Element.</br></br>



---------- JSX EXPRESSIONS ----------</br>

- A JSX Expression is any JavaScript code enclosed within Curly Brackets inside JSX Elements.</br>
- JSX Expression allow Dynamic Rendering Values, Conditional Evaluations and Calculations within a Component Structure.</br>



---------- REACT FRAGEMENT ----------</br>

-  It is a Feature to Written multiple elements from a react component by allowing us to group a list of children without adding an extra node to the DOM.</br>
