-----> STATE : <br/><br/>

- State is a Plane JavaScript Object which is used by REACT to Represent information about the component's current situation.<br/>
- State is a Data that changes over a time.<br/>
- State is a Internal Property which is defaultly present in Class Based Component (CBC).<br/>
- If you want to use state functionality in Function Based Component (FBC) then you can use "useState" Hook to manage state.<br/><br/>


-----> React Hooks : <br/><br/>

- Hooks are In-Built methods present in react.<br/>
- They can only be used inside function based component.<br/>
- Example : useState, useEffects<br/><br/>


-----> Rules Of Hooks :<br/><br/>

- Hook Should always we used inside a component.<br/>
- Hook must always must start with "use".<br/>
- Hook cannot be used inside Conditional Statements and Looping Statements.<br/><br/>


-----> useState :<br/><br/>

- useState is a REACT Hook that allow functional components to have state.<br/>
- It is used to store and update the values that changes over a time within a component.<br/>
- SYNTAX : let[state,setState] = useState(INITIAL_VALUE)<br/><br/>

-> state : It is a Variable Name which hold initial value.<br/>
-> setState : It is a Function to change the current value of a state. It's name can be anything.<br/><br/>
