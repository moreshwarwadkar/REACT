-----> STATE : <br/><br/>

- State is a Plane JavaScript Object which is used by REACT to Represent information about the component's current situation.<br/>
- State is a Data that changes over a time.<br/>
- State is a Internal Property which is defaultly present in Class Based Component (CBC).<br/>
- If you want to use state functionality in Function Based Component (FBC) then you can use "useState" Hook to manage state.<br/><br/>
