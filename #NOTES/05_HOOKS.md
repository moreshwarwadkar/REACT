-----> React Hooks : 

- Hooks are In-Built methods present in react.
- They can only be used inside function based component.
- Example : useState, useEffects


-----> Rules Of Hooks :

- Hook Should always we used inside a component.
- Hook must always must start with "use".
- Hook cannot be used inside Conditional Statements and Looping Statements.


-----> useState :

- useState is a REACT Hook that allow functional components to have state.
- It is used to store and update the values that changes over a time within a component.
- SYNTAX : let[state,setState] = useState(INITIAL_VALUE)

-> state : It is a Variable Name which hold initial value.
-> setState : It is a Function to change the current value of a state. It's name can be anything.
